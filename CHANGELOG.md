## 7.0.1 (2025-10-26)

## [0.0.1] - 2025-10-26

### Added

- Initial release of @capgo/capacitor-wechat
- WeChat installation detection
- WeChat OAuth authentication
- Content sharing (text, images, links, music, video, mini-programs)
- WeChat Pay integration
- Mini-program opening support
- Invoice selection functionality
- iOS implementation with Swift
- Android implementation with Java
- Web platform stub implementation
- TypeScript definitions for all APIs
- Comprehensive documentation and examples

### Notes

- This is a framework plugin that requires WeChat SDK integration
- Native WeChat SDK must be added to iOS and Android projects separately
- App registration on WeChat Open Platform is required
